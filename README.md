# 差勤智慧小助手

這是一個可安裝的網頁小程式（PWA）。請先選擇「公務人員」或「教師」，再輸入請假問題；系統會顯示依據兩份提供規則整理的答覆與條文出處。無法辨識或無法由規則判定的問題，會提示使用者逕洽人事單位。

## 使用與分享

1. 本機使用：開啟 `index.html`。
2. 手機安裝：將本資料夾發布至支援 HTTPS 的靜態網站空間後，以手機瀏覽器開啟；出現「安裝小程式」按鈕時即可加入主畫面。
3. 提供大家使用：將本資料夾的檔案上傳至學校或機關既有網站空間、GitHub Pages、Netlify 或 Cloudflare Pages 等靜態網站服務，即可分享該網址。請使用 HTTPS，才能啟用離線功能與安裝。

資料來源：

- `教師請假規則.pdf`（民國 114 年 10 月 8 日修正）
- `公務人員請假規則.pdf`（民國 114 年 9 月 18 日修正）
